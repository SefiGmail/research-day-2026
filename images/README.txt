LOGO INSTRUCTIONS
=================

Place your logo image file in this folder and name it:

    logo.png

Recommended dimensions: around 200 x 100 pixels (or larger, same aspect ratio).
The header is set to display it at 52px height; width scales automatically.

If no logo file is present, the site will display the faculty name as text instead.

Suggested sources:
  - Technion official branding: https://www.technion.ac.il/en/about/graphics-and-fonts/
  - ME Faculty website:         https://meeng.technion.ac.il/
